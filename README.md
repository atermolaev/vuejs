# Проект на VueJS

## Инсталяция проекта
```
npm install
```

### Запуск проекта в среде разработки
```
npm run serve
```

### Сборка продакшн версии проекта
```
npm run build
```

### Запуск линтера
```
npm run lint
```

### Изменение конфигурации
Смотри [описание конфигурации](https://cli.vuejs.org/config/).
